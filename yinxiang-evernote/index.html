我使用的App之印象笔记

发表于 2014-03-25 | 更新于 2019-06-09 | 分类于 软件

上一篇《关于情商的讨论》发布之后，朋友们的讨论非常热烈，大部分跟我持不同意见，不过我仍然很开心大家愿意花时间看完，然后留下自己的观点。我说过，每一条回复都让我心花怒放，这几天我是多么欢乐啊。朋友们的讨论也让我对于这个话题有了更多思考，媳妇儿问我要不要继续写“情商”，我说暂时不写了，等我再多想想，再多读点书，以后有机会再写。

继续介绍我手机上的App，要出场的是我使用频率最高的一款软件——印象笔记。

1 缘起

如果让我选择一款对工作和生活帮助最大的软件，去年是OneNote（微软Office里的一个笔记本软件），今年则是印象笔记（国际版叫Evernote）。它们解决的痛点是：随时随地记录、查找和阅读笔记，而且不用担心丢失。

大概五年前，我在安装Office 2010试用版的时候发现了OneNote，当时就爱不释手，把所有需要记录的东西，如工作计划、会议记录、项目方案、读书笔记、个人总结、日记、清单等等，都记在OneNote上。我建立了四个笔记本（项目、工作、个人、资料），每个笔记本里又建了很多子笔记本，然后是一个个的笔记。后来出现了Evernote，非常火，喜欢尝试的我自然也试用了，不过当时还是觉得OneNote的功能更强一些，最关键是跟Word的无缝对接，虽然没有手机版App（后来也有了），但是我平常随身携带笔记本电脑，也没有觉得不方便。

再后来Evernote国内版——印象笔记出来之后，我在手机反复安装卸载了很多次，终于在今年年初注册了一个印象笔记帐号，把Evernote中的笔记全部导入了印象笔记。印象笔记的免费用户每月可以上传60M，第一次导入笔记时可以申请临时调整到180M，我尝试申请了临时调整，竟然直接给我调整到了1G，我心想太好了，就连同OneNote里的笔记全部导过去，从此我纠结了好几年的网络笔记本服务选择问题终于尘埃落定：放弃已经使用了5年的Onenote，转投印象笔记。

2 印象笔记的优点

- 跨平台

Windows、Mac、Android、iOS都有客户端，还有网页版，而且体验都非常棒。OneNote也有Andriod的App，我先后下载了好几次，操作非常不便，长期不更新，超过500条笔记还要升级为收费用户。（几天前OneNote宣布全平台全免费并开放API。）

- 标签

可以为一个笔记设置不同的标签，还可以对标签分级。这样就不再需要创建很多笔记本，通过对标签的管理就可以达到类似文件夹分级管理的效果。

- 剪藏

Chrome浏览器的Evernote Web Clipper扩展可以一键保存页面到印象笔记中；Feedly中使用IFTTT可以做到点“Save for Later”按钮，会自动把那篇文章转存到印象笔记；小米手机上使用“发送”按钮可以直接把文字、图片、页面等内容发送到印象笔记。 - 稳定。原先使用OneNote和Evernote，时不时会出现无法同步的问题，这个对于同步控的我来说非常抓狂。使用印象笔记之后，同步速度和稳定性都很好，从来没有出现过无法同步的问题。

- 用户广泛

因为用的人多，所有很多人就会写如何更好使用印象笔记的文章，印象笔记自己的博客也会提供很多技巧，也有很多人在印象笔记上分享资料。我几乎没有看到过对印象笔记的负面评论。

- 其他

软件的功能非常多，我不想在此一一列举哪些功能，其他的就自己慢慢去找吧，请享受这个过程。

3 印象笔记的缺点

- 流量限制

OneNote上传流量没有限制，总的容量限制是Skydrive的容量为限7G，印象笔记免费用户每月上传流量60M，收费用户每月1G，我现在直接使用网页版，60M流量完全够用了。如果未来流量不够的话，我一定会花钱升级的。

- 编辑功能不强

OneNote是微软Office旗下的产品，有相对丰富的与Office统一的编辑器功能，我以前会先在OneNote中编辑好格式再导出成Word文档，之后的编辑工作就少了许多。印象笔记的编辑器功能比较简单，OneNote中可以用Tab键和回车键快速创建表格，而印象笔记则必须点创建表格才可以。现在我在笔记本上只是记录文字和想法，然后在Word中编辑格式，反而在记录的时候更加专注。

- 没有时间log

OneNote有个看起来非常牛的功能，就是把笔记中每一行的编辑时间都记录下来了，如果遇到需要回头查询时间时，则非常方便。印象笔记则没有这个功能，不过我几乎从来没有用过这个功能。

4 我用印象笔记做什么

- 项目记录

对每个项目建立一个标签，然后每一次的客户拜访、项目会议、项目方案、项目总结等都先用印象笔记记录下来，需要形成文件时再导入Word中编辑。

- 生活记录

读了哪些书、看了哪些电影、听了哪些课、去了什么地方、想写的博客主题、甚至去超市要买什么东西等，我都会记在印象笔记中。有一次听一个老师的课，非常喜欢他讲的东西，就在下面记录他讲的每一句话，老师问我怎么一直在玩电脑，我说我在记笔记，下课后把我笔记给发他之后，他说从来没见过这么详细的笔记。

- 写日记

记录每天的工作计划、工作总结、睡眠、天气、饮食、突然冒出的想法等。

- 资料库

在网上看到一些有价值的文章，使用剪藏功能保存到印象笔记中，然后用标签进行分类。
