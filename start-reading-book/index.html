开始读书

发表于 2010-02-08 | 更新于 2019-06-09 | 分类于 读书

去年发现李笑来老师的博客之后，就一直受益于它。读了他的《把时间当朋友》消除了我对时间管理的偏见，读了他对学习英语的见解，而最令我无法忘记的 是他在《把时间当朋友》讲到的一小段与母亲之间的故事：

很小的时候，我的母亲给我种下一棵种子。她叫我读书要一字不漏，并且还要在一字不漏的情况下读若干遍。至于我问她“有些书大概看看不就行了么”的时 候，她只是轻轻地说，“大概看看就行的书，你看它干什么？”说来也怪，一向不怎么听话的我，这话却记住了，连母亲当时的神情到现在也记得清清楚楚。

虽然比笑来老师听到此话晚了很多，我却也牢牢记住了这个道理。在我的读书和思考中，我越来越能体会它的所蕴含的深意。我不只在读书上求精而不求多， 也尝试在工作和生活中践行。

我开始删除Google Reader上的价值并不高的订阅，只保留那些我认为最有价值的；

我开始控制我在网络上盲目的浏览习惯，而是下载一些只听过书名或者作者却没有读过的书；

我开始放弃匆匆阅读或者零碎阅读，而是全身心的边读边思考；

而在不久之前，我竟然想开始对我读过的每一本书都留下一点我的见解，于是开通了QQ空间和豆瓣。

在久违了语文课之后，在习惯了接受零碎的信息和肤浅的知识之后，我开始重新审视自己，希望能在读好书的过程中发现真实的自己和真实的世界。

当我读王朔先生的《看上去很美》时，我惊讶原来几岁的孩子也可以作为小说的主角并用第一人称叙述生活；

当我读朱光潜先生的《谈美》时，我发现自己对于这个曾经在中学语文课本上出现的美学大师和他所谈的美学竟然一无所知；

当我听蒋勋先生讲《红楼梦》的时候，我觉得我以前不只白读了这本书，而且是白学了语文。

虽然这一切可能都有点晚，但是我想现在能够醒悟，能捡回多少算多少吧。

2010-2-8
